安装到 iPhone 主屏幕

正式使用建议部署到 GitHub Pages，再用 Safari 打开网址。

  1. 用 iPhone Safari 打开这个网页。
  2. 点击底部分享按钮。
  3. 选择“添加到主屏幕”。
  4. 名称保留“行动积分”，点击“添加”。

第一版使用本机存储，不要清除 Safari 网站数据。换手机前先导出 JSON 备份。
